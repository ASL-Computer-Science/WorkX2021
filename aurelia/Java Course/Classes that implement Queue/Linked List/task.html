Linked List

The Linked List data structure is a linear structure like the ArrayList. Each element in the structure is a separate object with a data part and an address part. Each element is called a node. Elements are linked using pointers and addresses. Like Queues elements are processed in FIFO order. However it has methods that allow the user to add an element to the front of the list and remove from the back.
