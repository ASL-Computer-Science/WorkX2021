Intro to Queue's

A Queue is a linear data structure similar to the ArrayList. Unlike ArrayLists Queue's follow FIFO (First in First Out) order. This means that items can only be added to the back of the queue and removed from the front. This is similar to how queues or checkout lines function in real life.
