Methods in Queue

Here are some methods from the Queue class:

  • add() - inserts the specified element at the back of the queue if possible, returns true upon sucess
  • offer() - inserts the element into the queue
  • remove() - throws an exception if there are no elements to take out
  • poll() - removes an element from the front of the Queue, returns null if no elements to take out
  • element
  • peek() - inspects the element at the front of the Queue, without removing it
  • size() - returns the size fo the queue
  • contains() - checks to see if the queue contains a certain element
