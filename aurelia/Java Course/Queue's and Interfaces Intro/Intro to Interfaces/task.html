Introduction to Interfaces

Now that we know that Queue is an interface it is important that we understand Interfaces before examining its uses and methods .

Interfaces are similar to classes because they contain methods and variables. However, methods declared in an interface are abstract, meaning they have no body. This means that Interface objects cannot be created. Classes that implement an interface can create objects but they must implement all of its methods.
